# container-scan-to-sarif

[![Go Report Card](https://goreportcard.com/badge/github.com/rm3l/container-scan-to-sarif)](https://goreportcard.com/report/github.com/rm3l/container-scan-to-sarif)

Converts [Azure Container Scan Action](https://github.com/Azure/container-scan#action-output) output to [Static Analysis Results Interchange Format (SARIF)](https://sarifweb.azurewebsites.net/), for an easier integration with [GitHub Code Scanning](https://docs.github.com/en/code-security/code-scanning/automatically-scanning-your-code-for-vulnerabilities-and-errors/about-code-scanning)
